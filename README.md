# Retail Sales Analytics

## 📌 Project Overview

This project analyzes retail sales data to identify key business insights, improve profitability, and support data-driven decision-making using SQL, Python, Excel, and Power BI.

---

## 🎯 Business Problem

The management observed that although sales appear to be growing, profits are not increasing as expected. The objective is to identify the factors affecting profitability and provide actionable recommendations.

---

## 🛠️ Tools

- SQL
- Python
- Excel
- Power BI
